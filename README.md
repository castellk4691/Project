# Prototype MVP

## What It Does
It gives singers cover ideas cover for TikTok posts by the user providing the song name, artist, and the asthetic/vibe they're going for. The bot gives 2 color palette options, typography suggestion, caption, and a short list of hashtags.

## How to Use
1. Type the song and artist name
2. Type the want you want
3. Click "Generate My Vibe".
4. Choose between the two color options
